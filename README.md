## 🚀 About

Welcome to the **Hacking & Cyber Security Learning Repository**! This collection of resources is designed to guide you through the fundamentals of ethical hacking and cybersecurity. Whether you're a beginner looking to start your journey or an intermediate learner wanting to deepen your knowledge, this repository has something for you.

### What You'll Learn
- How systems work (operating systems, networks)
- Common vulnerabilities and attack vectors
- Defensive strategies and security best practices
- Penetration testing methodologies
- Career paths in cybersecurity

---

## ✅ Prerequisites

Before diving into advanced topics, ensure you have a solid foundation in:

### 💻 Technical Skills

- **Basic Computer Literacy:** Understanding how computers work
- **Networking Fundamentals:** TCP/IP, DNS, DHCP, HTTP/HTTPS, ports
- **Operating Systems:** Strong knowledge of Linux (Ubuntu/Kali) and Windows
- **Command Line Proficiency:** Bash/PowerShell scripting
- **Programming Basics:** Python is highly recommended (for automation and tools)

### 🛠️ Environment Setup

- **Virtualization Software:** VMware Workstation or VirtualBox
- **Operating System:** Kali Linux (for hacking) and Windows/Linux (host)
- **Text Editor:** VS Code or Sublime Text
- **Browser:** Firefox/Chrome with developer tools

## 🎯 Topics Covered

### Phase 1: Fundamentals (Weeks 1-4)

- [ ] **Networking Basics**
  - [ ] OSI and TCP/IP models
  - [ ] IP addressing and subnetting
  - [ ] DNS, DHCP, HTTP, HTTPS
  - [ ] Common network protocols

- [ ] **Linux Fundamentals**
  - [ ] File system hierarchy
  - [ ] Permissions and users
  - [ ] Command-line basics (Bash)
  - [ ] Package management (apt, yum)

- [ ] **Windows Fundamentals**
  - [ ] Registry, file system, permissions
  - [ ] PowerShell basics
  - [ ] Active Directory overview

- [ ] **Programming Basics**
  - [ ] Python for security (requests, sockets)
  - [ ] Bash scripting basics

### Phase 2: Web Security (Weeks 5-10)

- [ ] **Web Technologies**
  - [ ] HTML, CSS, JavaScript
  - [ ] Client-side vs server-side
  - [ ] HTTP methods and headers

- [ ] **Common Vulnerabilities (OWASP Top 10)**
  - [ ] SQL Injection
  - [ ] Cross-Site Scripting (XSS)
  - [ ] Authentication vulnerabilities
  - [ ] Security misconfigurations
  - [ ] Sensitive data exposure

- [ ] **Web Tools**
  - [ ] Burp Suite (essential!)
  - [ ] OWASP ZAP
  - [ ] Browser developer tools

### Phase 3: Penetration Testing (Weeks 11-16)

- [ ] **Reconnaissance**
  - [ ] Passive reconnaissance (OSINT)
  - [ ] Active reconnaissance (nmap, dnsenum)
  - [ ] Footprinting techniques

- [ ] **Scanning & Enumeration**
  - [ ] Nmap (port scanning, service detection)
  - [ ] SMB, SNMP, LDAP enumeration
  - [ ] Vulnerability scanning

- [ ] **Gaining Access**
  - [ ] Password attacks (brute force, dictionary)
  - [ ] Exploiting vulnerabilities (Metasploit)
  - [ ] Phishing and social engineering
  - [ ] Wireless attacks

- [ ] **Post-Exploitation**
  - [ ] Privilege escalation
  - [ ] Lateral movement
  - [ ] Maintaining access
  - [ ] Covering tracks

### Phase 4: Advanced Topics (Weeks 17+)

- [ ] **Malware Analysis**
  - [ ] Static analysis
  - [ ] Dynamic analysis
  - [ ] Reverse engineering basics

- [ ] **Digital Forensics**
  - [ ] Incident response
  - [ ] Evidence collection
  - [ ] Chain of custody

- [ ] **Cloud Security**
  - [ ] AWS/Azure security
  - [ ] Container security
  - [ ] Serverless security

- [ ] **IoT Security**
  - [ ] Common IoT vulnerabilities
  - [ ] Firmware analysis
  - [ ] Device hacking

## 🛠️ Tools You'll Use

### 🔧 Network Tools

| Tool | Purpose |
|------|---------|
| **Nmap** | Network discovery and port scanning |
| **Wireshark** | Network traffic analysis |
| **Aircrack-ng** | Wireless security testing |
| **Nessus** | Vulnerability scanning |

### 🔧 Web Security Tools

| Tool | Purpose |
|------|---------|
| **Burp Suite** | Web application security testing (essential) |
| **OWASP ZAP** | Free, open-source alternative |
| **SQLMap** | Automatic SQL injection testing |
| **Nikto** | Web server scanner |

### 🔧 Exploitation & System Tools

| Tool | Purpose |
|------|---------|
| **Metasploit Framework** | Penetration testing framework |
| **Hashcat** | Password recovery |
| **John the Ripper** | Password cracking |
| **Hydra** | Brute-force login attacks |
